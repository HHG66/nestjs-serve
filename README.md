# 简介

nestjs+MongoDB完成的项目

## 安装

```bash
$ npm install
```

## 配置

配置文件在`src\config`下，分别是日志、数据库、登录认证的配置，修改对应的配置后启动

## 构建运行

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
